Egyptian Wars

A digital version of Egyptian Rat Screw — 1v1 against an AI bot, built from scratch in HTML, CSS, and JavaScript

JavaScript Web Dev Apr 2026

About

Egyptian Wars (also known as Egyptian Rat Screw) has always been my favorite card game. After searching the internet for a digital version without success, I decided to build it myself from scratch using HTML, CSS, and JavaScript.

The game is a 1v1 battle against an AI bot where players race to collect all 52 cards. The core mechanics revolve around identifying specific patterns — Doubles, Sandwiches, Marriages, Divorces, and Top & Bottoms — to slap the pot and win the pile. I also implemented the complex "Challenge" system, where face cards (Jack, Queen, King, Ace) force the opponent to play a specific number of cards to stay in the game.

Technical Implementation

The game's engine is built on a custom JavaScript state machine. Boolean flags (canPlayerPlay, canBotSlap, etc.) enforce turn integrity and block illegal moves. The most significant technical hurdle was managing the intersection of bot latency and real-time user input — to keep the game feeling fair, I engineered a system using setTimeouts that lets the bot "react" at varying delays based on difficulty. If a player slaps before the bot's timer clears, the scheduled action is instantly cancelled via clearTimeout to prevent race conditions.

Implementing the Challenge system was equally complex — a dedicated resolveChallenge() function handles back-to-back face cards, depleted attempt counters, mid-challenge slap opportunities, and challenger role switching, all while keeping the boolean state consistent across both players.

Design & UX

The interface uses Bootstrap and CSS Grid in a 3×3 layout that keeps the action center-column with the burn pile, slap status, and game state arranged around it. Card images are pulled dynamically from the Deck of Cards API via template literals. To optimize for high-speed gameplay, Play is mapped to J and Slap to SPACE — a layout chosen for ergonomic comfort and reaction speed.

The pot is styled so the top three cards are always slightly fanned, letting players spot patterns at a glance without the UI feeling cluttered. The result is a fast-paced, responsive browser game that finally brings my favorite card game to the digital space.
